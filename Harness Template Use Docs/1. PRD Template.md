# PRD Template

이 문서는 범용 Harness Template을 실제 프로젝트용 하네스로 변환하기 전에 작성하는 제품 요구사항 원본입니다.

PRD는 제품 판단의 기준이고, 하네스 문서는 AI와 개발자가 안전하게 구현하기 위한 실행 기준입니다. PRD를 작성한 뒤 필요한 내용을 `docs/PRODUCT_CONTEXT.md`, `ARCHITECTURE.md`, `docs/TESTING.md` 등으로 분배합니다.
