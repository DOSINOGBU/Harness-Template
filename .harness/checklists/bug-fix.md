# Checklist: Bug Fix

- [ ] 문제를 한 문장으로 요약했습니다.
- [ ] 재현 조건을 확인했습니다.
- [ ] 수정 전 실패 로그, 화면, 입력, 환경 정보를 보존했습니다.
- [ ] 가능한 원인을 3~5개로 좁히고 각 근거를 적었습니다.
- [ ] 가장 가능성 높은 원인을 코드, 로그, 테스트 중 하나 이상으로 확인했습니다.
- [ ] 최소 수정으로 고쳤습니다.
- [ ] 관련 없는 리팩터링이나 구조 변경을 섞지 않았습니다.
- [ ] 수정 전 실패와 수정 후 성공을 검증했습니다.
- [ ] 남은 리스크나 확인하지 못한 부분을 기록했습니다.
