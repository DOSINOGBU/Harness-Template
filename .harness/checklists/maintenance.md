- [ ] 발견사항을 `safe cleanup`, `needs review`, `ignore for now`로 분류했습니다.
- [ ] 자동 수정 금지 항목이 포함되지 않았습니다.
- [ ] 기능 변경, 버그 수정, 리팩터링과 섞지 않았습니다.
- [ ] 인덱스 누락은 README 보강 후보로 분리했습니다.
- [ ] 오래된 active plan은 완료, 보류, 폐기 중 하나로 분류했습니다.
- [ ] generated 문서는 직접 수정하지 않고 재생성 필요 여부를 기록했습니다.
- [ ] 반복된 실수는 `docs/LEARNING_LOG.md`에 기록하거나 기존 항목을 갱신했습니다.
- [ ] 정리 후 기본 하네스 검증을 다시 실행했습니다.
- [ ] 남은 드리프트와 사람이 확인해야 할 항목을 보고했습니다.
