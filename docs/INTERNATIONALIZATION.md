# Internationalization

언어, 로케일, 시간대, 숫자 형식 기준을 기록합니다.

## Policy

- 사용자에게 보이는 시간은 명시된 timezone 기준으로 표시합니다.
- 날짜/시간은 상대 표현만 쓰지 않고 필요한 곳에 절대 날짜를 함께 둡니다.
- 하드코딩된 UI 문구는 프로젝트의 번역 정책을 따릅니다.
- 숫자, 통화, 단위는 사용자의 locale 기준을 고려합니다.
