# Glossary

도메인 용어를 AI가 추측하지 않도록 정의합니다.
비슷해 보이는 개념의 차이를 반드시 적습니다.

## Terms

| 용어 | 의미 | 혼동 금지 |
|---|---|---|
| User | 서비스를 사용하는 계정 주체 | 방문자, 세션과 혼동하지 않음 |
| Session | 로그인 또는 작업 흐름의 일시적 상태 | User 자체가 아님 |
| Project | 사용자가 관리하는 작업 단위 | Workspace와 구분 필요 |

## Naming Rules

- 코드 이름은 이 문서의 용어를 우선 사용합니다.
- 새 용어가 필요하면 먼저 이 문서에 정의합니다.
- 같은 의미에 여러 단어를 섞어 쓰지 않습니다.
